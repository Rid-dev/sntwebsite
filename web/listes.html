Les listes

On retrouve deux types de listes :

Les listes numérotées

  1. Le premier élément
  2. Le deuxième élément
  3. Le troisième élement
  4. Le quatrième ....

Les listes non numérotées

  • Un élément
  • Un autre élément
  • Encore un autre élément
  • C'est bon, on a compris !
