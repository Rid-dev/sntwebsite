Code de base pour commencer une page HTML


Pour débuter une page html, on pourra utiliser le code suivant. Attention à remplacer le titre (qui apparait dans l'onglet) !

<!DOCTYPE html>
	
<html>

  <head>
    <meta charset="utf-8">
    <title>Titre de la page</title>
    <link rel="stylesheet" href="style.css">
  </head>
  
  <body>
    On écrit ici le contenu de la page...
  </body>
  
</html>
